# 管線工程預算估算器 (Pipeline Budget Estimator)

## 描述

基於台灣自來水公司管線工程預算書資料，提供1000mm管線單位長度施工單價查詢，以及各管徑預算推估功能。

## 適用情境

- 查詢1000mm管線各施工項目單價
- 依據參考預算書推估不同管徑工程預算
- 計算管線工程主要項目數量與金額
- 分析管線工程成本結構

## 管徑預算推估公式

### 基準參數（1000mm管線）

- 參考總長度：4,860 M（日間施工）
- 參考總預算：245,930,000 元（發包部分）
- 參考單價：約 50,603 元/公尺（含所有費用）

### 推估係數

依據經驗，不同管徑的相對成本係數約為：

| 管徑(mm) | 成本係數 | 單位長度推估單價(元/M) |
|---------|---------|---------------------|
| 150 | 0.25 | 12,651 |
| 200 | 0.30 | 15,181 |
| 250 | 0.35 | 17,711 |
| 300 | 0.40 | 20,241 |
| 400 | 0.50 | 25,302 |
| 500 | 0.60 | 30,362 |
| 600 | 0.70 | 35,422 |
| 700 | 0.80 | 40,482 |
| 800 | 0.90 | 45,543 |
| 1000 | 1.00 | 50,603 |
| 1200 | 1.25 | 63,254 |
| 1350 | 1.45 | 73,374 |
| 1500 | 1.65 | 83,495 |

### 預算推估公式

```
總預算 = 管線長度(M) × 管徑係數 × 基準單價(50,603元/M) × 調整因子

其中調整因子：
- 地質條件：軟質土壤 1.0，礫石層 1.15，岩盤 1.30
- 施工環境：郊區 1.0，市區 1.20，高鐵/捷運敏感區 1.35
- 路面類型：一般道路 1.0，高速公路 1.25，鐵路穿越 1.40
```

## 計算範例

### 範例1：查詢1000mm管線單價

```
輸入：查詢1000mm管線埋設單價
輸出：
- 直管埋設單價：890元/M
- 推進工法單價：80,000元/M（含接頭）
- 延性鑄鐵管材料：95,800元/M（含配件）
```

### 範例2：推估800mm管線500公尺預算

```
輸入：推估800mm管線500公尺，市區道路，一般土壤
計算：
- 管徑係數：0.90
- 環境係數：1.20（市區）
- 地質係數：1.00（一般土壤）
- 預算 = 500 × 0.90 × 50,603 × 1.20 = 27,325,620元

輸出：
預估總預算約 2,733萬元（未稅）
- 管線主體：約 2,280萬元
- 路面修復：約 228萬元
- 臨時工程：約 137萬元
- 雜項費用：約 88萬元
```

### 範例3：計算CLSM回填數量

```
輸入：計算1000mm管線4,952M的CLSM回填量
計算：
- 非公路局路段：16,596 M³
- 公路局路段：3,335 M³
- 加深段：513 M³
- 穿越高鐵段：109 M³
- 總計：20,553 M³

單價：1,100元/M³
總價：22,608,300元
```

## 注意事項

1. 以上單價基於嘉義科學園區供水北線工程-管(一)預算書
2. 實際單價會因物價指數、施工條件、工期等因素調整
3. 推進工法單價遠高於明挖工法，選擇時需綜合考量
4. 路面修復費用會因道路等級、瀝青厚度而異
5. 臨時擋土設施費用會因地質條件、開挖深度而調整
