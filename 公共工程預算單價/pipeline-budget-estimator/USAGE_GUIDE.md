## 參數說明

### 地質條件
- **軟質土壤**：係數 1.0（預設）
- **礫石層**：係數 1.15
- **岩盤**：係數 1.30

### 施工環境
- **郊區**：係數 1.0（預設）
- **市區**：係數 1.20
- **高鐵敏感區**：係數 1.35
- **捷運敏感區**：係數 1.35

### 路面類型
- **一般道路**：係數 1.0（預設）
- **高速公路**：係數 1.25
- **鐵路穿越**：係數 1.40

## 計算範例

### 範例 1：市區 800mm 管線
```
輸入：推估 800mm 管線 500 公尺，市區，一般土壤

計算：
- 管徑係數：0.90
- 環境係數：1.20（市區）
- 地質係數：1.00（一般土壤）
- 總調整因子：1.20

預算 = 500 × 0.90 × 50,603 × 1.20 = 27,325,620 元（未稅）
```

### 範例 2：高鐵敏感區 1000mm 管線
```
輸入：推估 1000mm 管線 300 公尺，高鐵敏感區

計算：
- 管徑係數：1.00
- 環境係數：1.35（高鐵敏感區）
- 總調整因子：1.35

預算 = 300 × 1.00 × 50,603 × 1.35 = 20,494,215 元（未稅）
```

### 範例 3：CLSM 回填計算
```
輸入：計算 1000mm 管線 4,952M 的 CLSM 回填量

計算：
- 非公路局路段：16,596 M³
- 公路局路段：3,335 M³
- 加深段：513 M³
- 穿越高鐵段：109 M³
- 總計：20,553 M³

總價 = 20,553 × 1,100 = 22,608,300 元
```

## 注意事項

1. **單價基準**：所有單價基於嘉義科學園區供水北線工程-管(一)預算書
2. **物價調整**：實際單價會因物價指數、施工條件、工期等因素調整
3. **工法差異**：推進工法單價（80,000 元/M）遠高於明挖工法（890 元/M）
4. **路面等級**：路面修復費用會因道路等級、瀝青厚度而異
5. **地質影響**：臨時擋土設施費用會因地質條件、開挖深度而調整

## 參考資料

- **工程名稱**：嘉義科學園區供水北線工程-管(一)
- **工程編號**：WR11405010027
- **施工總長度**：4,952 M（明挖）+ 358 M（推進）
- **總預算**：258,226,500 元（含稅）
