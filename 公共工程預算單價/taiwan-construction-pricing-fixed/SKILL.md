# 台灣公共工程單價庫 (Taiwan Construction Pricing Database)

## 技能概述

本技能提供台灣公共工程相關的單價查詢、材料價格、物價指數等資料，整合行政院公共工程委員會、各縣市政府開放資料平台等多個資料來源，協助工程預算編列與估價作業。

## 快速使用指南

### 查詢大宗資材價格

```python
import pandas as pd

# 讀取 CSV 資料
df = pd.read_csv('https://pcic.pcc.gov.tw/pwc-web/api/service/opendata-file/document/大宗資材及其漲跌幅彙整表.csv')

# 查詢特定材料價格
steel_price = df[df['調查項目'].str.contains('鋼筋')]
print(steel_price[['調查項目', '調查地區', '114.8價格(A)']])
```

## 工程預算估算方法

### 基本公式

```
工程預算 = Σ(各項工程量 × 單價) × (1 + 管理費率) × (1 + 利潤率) × 物價調整係數
```

### 物價調整係數計算

```
調整係數 = (當期物價指數 / 基期物價指數)
```

### 範例計算

假設某工程項目：
- 混凝土用量：100 立方公尺
- 單價：2,500 元/立方公尺
- 管理費率：15%
- 利潤率：8%
- 物價調整係數：1.05

```
直接工程費 = 100 × 2,500 = 250,000 元
含管理費 = 250,000 × 1.15 = 287,500 元
含利潤 = 287,500 × 1.08 = 310,500 元
調整後預算 = 310,500 × 1.05 = 326,025 元
```

## 注意事項

1. **資料更新時間**：各資料集更新頻率不同，使用時請注意資料時間戳記
2. **區域差異**：材料價格因區域而異，請依工程所在地選擇適當資料
3. **市場波動**：單價資料僅供參考，實際編列預算應核實市場行情
4. **授權條款**：資料採用「政府資料開放授權條款-第1版」，使用時請遵循相關規定
