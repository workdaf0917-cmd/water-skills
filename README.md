# Water Skills

台灣水利與工程相關 Claude Skills 集合

## 包含技能

### 水利署相關
- **agongdian-reservoir-realtime** - 阿公店水庫即時水情
- **reservoir-daily-operations** - 水庫每日營運狀況
- **reservoir-historical-storage** - 水庫歷史蓄水量
- **river-flow-monitoring** - 河川流量監測
- **wra-basin-rainfall** - 集水區降雨量
- **wra-water-rights** - 水權統計調查

### 工程專業
- **engineering-params-extractor** - 工程參數與法規提取
- **公共工程預算單價** - 公共工程預算單價查詢

### 環境監測
- **reservoir-water-quality** - 水庫水質監測
- **環保署水庫水質監測API** - 環保署水質資料

### 統計與主計
- **主計/nstat-query-system** - 主計處統計查詢
- **主計/nstat-salary-query** - 薪資統計查詢

## 使用方法

這些技能專為 Claude Code CLI 設計。將需要的技能資料夾複製到你的專案 `.claude/skills/` 目錄下即可使用。
