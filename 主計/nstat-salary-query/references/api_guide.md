### 統計期
- 資料範圍：1973年（民國62年）至今
- 年資料 id 格式：`2024`
- 月資料 id 格式：`2024-M1` ~ `2024-M12`
- 數值單位：新台幣元

## API-JSON 回應結構

```
{
  "meta": { ... },
  "data": {
    "dataSets": [{
      "series": {
        "0:0": { "observations": { "0": [值], "1": [值], ... } },
        "0:1": { ... },
        ...
      }
    }],
    "structure": {
      "dimensions": {
        "series": [ {維度定義} ],
        "observation": [ {時間期定義} ]
      }
    }
  }
}
```

### Series Key 對應
- 格式：`{產業別index}:{性別index}`（皆為 0-based）
- 例：`"0:0"` = 工業及服務業 + 合計
- 例：`"3:1"` = 製造業 + 男

### Observation Key 對應
- 格式：0-based 索引
- 對應 `structure.dimensions.observation[0].values` 中同索引項目

## 技術限制
1. URL 長度限制 1,000 字元
2. 帶維度篩選之 URL 格式目前回傳 HTML 而非 JSON，需下載完整資料集後客戶端篩選
3. 匿名查詢，無需 API Key
4. 資料更新頻率依主計總處公告（通常每月）
