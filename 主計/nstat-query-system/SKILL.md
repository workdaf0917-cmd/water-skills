# 台灣總體統計資料庫查詢系統

## 描述

透過行政院主計總處「總體統計資料庫 API」查詢各類官方統計數據，涵蓋國民所得、物價、勞工、金融、人口、教育、衛生等 **20 個統計領域、254 個資料表**，支援年度、季度、月度數據查詢。

## 觸發條件

當用戶需要：
- 查詢台灣官方統計數據（GDP、CPI、薪資、失業率、人口等）
- 分析經濟指標時間序列
- 取得特定產業/領域統計資料
- 比較不同年度/月度統計數據
- 列出某統計領域可用的資料表

## API 基本資訊

### Base URL
```
https://nstatdb.dgbas.gov.tw/dgbasAll/webMain.aspx?sdmx/
```

### 驗證方式
匿名查詢，無需 API Key。URL 長度限制 1,000 字元。

### 查詢模式

#### 模式 1：完整資料集下載（推薦）
```
https://nstatdb.dgbas.gov.tw/dgbasAll/webMain.aspx?sdmx/{功能代碼}
```
- 回傳完整 API-JSON，包含所有維度與全部統計期間
- 客戶端篩選產業別、性別、時間等維度

#### 模式 2：帶維度篩選查詢
```
https://nstatdb.dgbas.gov.tw/dgbasAll/webMain.aspx?sdmx/{功能代碼}/{維度篩選}.{週期}&startTime={起始}&endTime={結束}
```
- 維度間以 `.` 分隔，同維度項目以 `+` 串接
- 週期：`A`（年）、`Q`（季）、`M`（月）
- 時間格式：`2024`（年）、`2024-M1`（月）、`2024-Q1`（季）
- **注意**：部分資料表此模式回傳 HTML 而非 JSON，此時請改用模式 1

## API-JSON 回應格式

```json
{
  "meta": { "prepared": "查詢時間", "sender": {...} },
  "data": {
    "dataSets": [{
      "series": {
        "dim0_idx:dim1_idx": {
          "observations": { "time_idx": [值], ... }
        }
      }
    }],
    "structure": {
      "dimensions": {
        "series": [ {"name": "維度名", "keyPosition": 0, "values": [...]} ],
        "observation": [ {"name": "統計期", "values": [...]} ]
      }
    }
  }
}
```

### Series Key 規則
- 格式：`{dim0_index}:{dim1_index}:...`（皆 0-based）
- 維度順序對應 `structure.dimensions.series` 陣列順序

### Observation Key 規則
- 0-based 索引，對應 `structure.dimensions.observation[0].values` 同索引項目

## 工作流程

### Step 1：理解查詢意圖
- 從用戶自然語言識別統計領域與資料表
- 使用「常用關鍵字對照」或 `references/function_codes.json` 模糊匹配
- 提取時間範圍與週期需求

### Step 2：查找功能代碼
- 精確匹配：直接查找 function_codes.json
- 模糊匹配：找出最接近的 3 個資料表，詢問用戶確認

### Step 3：下載並解析資料
- 呼叫 `scripts/query_nstat.py` 執行查詢
- 解析 API-JSON 結構，提取維度定義與數據

### Step 4：篩選與格式化
- 依用戶需求篩選特定維度項目與時間範圍
- 輸出 Markdown 表格或 JSON

## 腳本使用方式

```bash
# 查詢特定資料表（完整資料集，客戶端篩選）
python scripts/query_nstat.py --code A046201010 --start 2024-M1 --end 2024-M12

# 按功能代碼查詢，指定維度
python scripts/query_nstat.py --code A046201010 --dim0 1 --dim1 1 --start 2020 --end 2024

# 用關鍵字搜尋資料表
python scripts/query_nstat.py --search "薪資"

# 列出特定領域所有資料表
python scripts/query_nstat.py --domain "勞工統計"

# 列出所有領域
python scripts/query_nstat.py --list-domains

# 查看資料表維度結構
python scripts/query_nstat.py --code A046201010 --describe

# 輸出 JSON 格式
python scripts/query_nstat.py --code A046201010 --start 2024 --end 2024 --format json
```

## 錯誤處理

| 錯誤情境 | 處理方式 |
|---------|---------|
| 找不到資料表 | 列出最相似的 3 個選項供確認 |
| 時間範圍無資料 | 回報可查詢範圍，建議調整 |
| API 連線失敗 | 重試 1 次（間隔 2 秒），仍失敗回報錯誤 |
| API 回傳 HTML | 自動切換至完整資料集模式 |
| 維度參數錯誤 | 顯示該資料表可用維度與項目清單 |

## 限制與注意事項

1. 數據依主計總處公布時程更新（月資料約次月底、年資料約次年 2~3 月）
2. 完整資料集下載量可能較大（數 MB），建議適度快取
3. 不支援跨表查詢，每次僅能查詢單一功能代碼
4. 數值單位依各資料表定義（千人、新台幣元、%、指數等），輸出時須標註
5. 必須標註「資料來源：行政院主計總處」
